# yudao

## 一、快速启动

### 1.1 前端

```markdown
# clone 项目
git clone https://github.com/yudaocode/yudao-ui-admin-vue3.git
# 安装依赖
pnpm install
# 项目启动
pnpm run dev
```

### 1.2 后端

导入项目
